# wasmprinter

**A [Bytecode Alliance](https://bytecodealliance.org/) project**

A Rust parser for printing a WebAssembly binary in the [WebAssembly Text format
(WAT)][wat]

[wat]: http://webassembly.github.io/spec/core/text/index.html
